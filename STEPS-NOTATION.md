### Iniciando projeto:

1. Pra iniciar o projeto react com todas as pastas:
```
npm create vite@latest . -- --template react-ts
```

2.Instalando as dependecias do projeto
```
npm install
```

3.executar localmente
```
npm run dev
```

4. deployar no github pages
```
git push origin main
```

optional:
```
nvm use
```
